On me demande parfois... où trouver des développeuses(eurs) Python

  • On-me-demande-parfois
  • Python
June 29, 2015

Dernière mise à jour : 30 octobre 2018

Les événements où rencontrer des candidat(e)s

  • les conférences annuelles PyCon FR et PyParis
  • le meetup Paris.py ou celui de votre ville

  • les événements dédiés au recrutement :

    • RemixJobs Day
    • TechDoors

Les sites où diffuser une annonce

  • les job boards dédiés à Python :

    • le job board de l'AFPy (Association Francophone Python)
    • le job board de la PSF (Python Software Foundation)
    • le Free Python Job Board

  • les job boards français généralistes :

    • RemixJobs
    • Human Coders
    • Choose Your Boss

  • les sites internationaux :

    • le thread mensuel Who is Hiring? sur Hacker News
    • Stack Overflow Careers
    • GitHub Jobs
    • Jobs in Tech
    • LinkedIn Jobs

  • si c'est pour faire du Django :

    • Django Gigs
    • Django Jobs

  • si vous êtes une startup :

    • @frstartupjobs sur Twitter
    • AngelList Jobs
    • Startup Developer Jobs
    • le job board de Paris & Co (si vous êtes dans un de leurs incubateurs et pépinières)
    • le job board de NUMA
    • le job board de Station F

  • si vous utilisez des logiciels libres :

    • Lolix

  • si vous êtes ouverts au télétravail :

    • Remote OK
    • Remotive
    • We work remotely

  • enfin, si vous êtes une licorne :

    • UnicornHunt
